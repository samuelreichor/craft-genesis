## Features

- Import Sites, Entry Types, Sections, Filesystems and Assets.
- CSV validation after upload.
- Queue Jobs for handling the heavy lifting.
- Support for all major settings you can make.

## Requirements

This plugin requires Craft CMS 5.0.0 or later, and PHP 8.2 or later.
